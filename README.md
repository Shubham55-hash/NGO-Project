## ✨ Features

- **Responsive Design** — Adapts seamlessly across desktop, tablet, and mobile
- **Dark / Light Mode** — Persistent theme preference saved to `localStorage`
- **Scroll Animations** — Cinematic reveal transitions as sections enter view
- **Testimonial Carousel** — Touch, click, and keyboard navigable with ARIA support
- **Volunteer Modal** — Form with real-time validation and toast confirmation
- **Email Capture** — Shake-on-error validation, success fade-in, and layout preservation
- **Premium Typography** — Playfair Display + Manrope + Inter type system
- **Glassmorphism & Ambient Glows** — Editorial-quality visual polish
- **Accessible Interface** — ARIA labels, keyboard navigation, focus traps in modals

## 🎯 Project Goal

Design and build a visually compelling, emotionally meaningful NGO website that reflects the mission of the She Can Foundation — while maintaining strong frontend fundamentals: accessibility, responsiveness, and clean maintainable code.
